Project Title. Birthday Paradox

Motivation of the project The Birthday Paradox is the conudrum where two people see if they have the same birthday. It happens more often than you would think as there are only 365 days in a year and billions of people on Earth! We have made a program that makes a list of birthdays and sees if any match. We have split the project into 4 main files to seperate the work into clean code principles.

Functional requirements/what the project does Our program generates a list of unique (or maybe the same) birthdays. We used enums (a type of object) to classify our birthdays. It then runs through a number of trials (creating new birthdays), and then may find a matching birthday, and show the percentage. We have implemeneted tests to assure that all methods are working and that our code runs and sees how real the birthday paradox is!
